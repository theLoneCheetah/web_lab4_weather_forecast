# Приложение для прогноза погоды

[Деплой на GitHub Pages](https://thelonecheetah.github.io/web_lab4_weather_forecast/)

## Задача
* Создать веб-приложения с прогнозом погоды, где будет возможность отображения прогноза по выбранным городам на ближайшие три дня.
* Веб-приложение должно запрашивать данные о погоде у публичного API
* При появлении на странице должен запрашиваться доступ к геоданным пользователя, иначе ввод должен быть вручную.
* Не использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Предоставить deploy проекта.

## Реализованные функции
* В качестве публичного API был выбран [open-meteo](https://open-meteo.com/), предоставляющий удобный синтаксис GET-запросов для получения погоды по координатам городов и получения данных самих городов.
* Базовый макет страницы написан на HTML. Дизайе адаптирован также под мобильные устройства.
* Приложение поддерживает хранение нескольких городов и показ прогноза погоды по ним на три дня.
* Реализованы функции обновления данных, добавления и удаления города (кроме того, который считается выбранным или определённым текущим местоположением).
* Ввод города осуществляется через выпадающий список из вариантов с запросом к API.
* При обновлении/запросе данных отображается спиннер загрузки, при ошибках выводятся соответствующие информационные сообщения.
* Данные о выбранных городах (кроме обновляемого текущего местоположения) сохраняются и подгружаются из local storage.
